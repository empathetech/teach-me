This portfolio

An Empathetech community project.

Empathetech is a community for software engineers — built around the idea that empathy is central to our success. This teach-me hub is one of our community spaces, for members who want to teach themselves something specific and design a curriculum around it.

Each course on this hub is the work of an individual community member, named on their course's pages. The architecture, voice, and design decisions of every course are the designer's — not the AI's. Claude Code is the collaborative writing and engineering partner: drafting prose, writing interaction logic, iterating on visual treatments, all under the designer's direction. We name both — every page, every time — because honesty about how AI fits into the work is part of using it ethically. Every factual claim in any lesson is cited and traceable; see that lesson's 📚 Sources section or the course's Credits page.

Curricula are scaffolded with the open-source /teach-me Claude Code skill, which any community member can use to start their own course.
