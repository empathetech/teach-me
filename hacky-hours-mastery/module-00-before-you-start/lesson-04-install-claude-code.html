Install Claude Code

Two steps: get a paid Claude plan, then install Claude Code. There's a no-terminal way (a desktop app you download like any program) and a terminal way (one command). Pick whichever feels comfortable — both end with the same thing running on your computer.

8 min read hands-on

Step 1 — get a Claude account and plan

  1. Go to claude.ai and create an account (or sign in).
  2. Upgrade to a paid plan — Pro is the usual starting point. Remember from Lesson 2: the free plan can't run Claude Code.
Why a plan first? Claude Code signs in with your Claude account. If the account isn't on a paid plan (Pro, Max, Team, Enterprise, or a pay-as-you-go Console account), Claude Code won't let you in. Setting up the plan now saves a stumble later.

Step 2 — install Claude Code

Choose the path that suits you. They lead to the same place.

The gentle way: the desktop app (no terminal)

Anthropic offers a desktop app with a graphical interface, so you can use Claude Code without living in the terminal. Download it for macOS or Windows, install it like any other program, and open it.

Heads up: even with the desktop app, installing the Hacky Hours skill in the next lesson uses one terminal command. That's why we made friends with the terminal in Lesson 3 — you'll need it just that once. Everything after can stay in the app if you prefer.

If something goes wrong

Two friendly fallbacks:

  • Run claude doctor in the terminal — it checks your install and points at fixes.
  • See Anthropic's install & login troubleshooting guide.
