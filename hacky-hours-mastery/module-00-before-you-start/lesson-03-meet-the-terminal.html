Meet the terminal

The terminal is a plain window where you type a command and press Enter instead of clicking buttons. That's the whole idea. By the end of this lesson you'll have opened one and run a harmless command — and it'll feel like no big deal, because it isn't.

6 min read hands-on

What it actually is

You normally tell your computer what to do by clicking. The terminal lets you tell it what to do by typing — one line at a time. People who build software use it because typing a command is often faster and more precise than hunting through menus.

It's also called the "command line," the "command prompt," the "shell," or the "console" — all the same idea. It looks bare because it is: a prompt, your command, and the answer.

You can't easily break anything by opening it Opening the terminal and typing the commands in this course is safe. The risky commands are ones that delete things — and we'll never ask you to run one of those blind. When you're unsure what a command does, you can always ask Claude before you run it.

How to open it

It depends on your computer:

On a Mac

  1. Press ⌘ Command + Space to open Spotlight search.
  2. Type Terminal and press Enter.
  3. A window opens with a prompt. That's it — you're in.

On Windows

  1. Click the Start menu and type PowerShell.
  2. Click Windows PowerShell (the blue one) to open it.
  3. A window opens with a prompt that starts with PS C:\. You're in.
PowerShell vs CMD (Windows only): Windows has two of these. PowerShell's prompt starts with PS C:\; the older "Command Prompt" (CMD) shows just C:\ with no PS. The install commands in Lesson 4 are written for PowerShell, so open that one.

How to run a command

You type the command, then press Enter. That's the entire ritual. To use a command from this course, you don't even need to type it — you can copy it from the lesson and paste it in:

  • Mac: paste with ⌘ Command + V.
  • Windows PowerShell: paste with Ctrl + V, or just right-click.

What you just learned

  • The terminal is a window where you type commands and press Enter.
  • Open it via Spotlight (Mac) or by searching PowerShell (Windows).
  • You can copy commands from a lesson and paste them in — no typing required.
  • Opening it and running course commands is safe; ask Claude if you're ever unsure.
