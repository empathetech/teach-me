Install the Hacky Hours skill

One more command and you're ready. With Claude Code installed, you just add the Hacky Hours skill on top — then type /hacky-hours and you're in. Let's finish the setup.

6 min read hands-on

Step 1 — run the install command

Open your terminal (Lesson 3), pick your system below, and paste the command. This downloads the Hacky Hours skill and places it where Claude Code can find it.

🛠️ Pick your computer
Same careful habit as last lesson This command runs an installer from the Hacky Hours project's official GitHub. It's open-source, so you can read exactly what it does at the repository before running it. As always: only run install commands from sources you trust, and if anything looks off, stop and ask.

Step 2 — restart Claude Code

Skills are picked up when Claude Code starts. So close Claude Code and open it again (quit the terminal session with Ctrl + C or by closing the window, then run claude again — or just reopen the desktop app).

Step 3 — check it's there

Inside Claude Code, type:

If you see the Hacky Hours help message — a list of commands like step, team, adopt, and audit — the skill is installed and working. That's the finish line for setup. 🎉

Don't see it? Make sure you fully restarted Claude Code after installing. If it's still missing, re-run the install command and watch for any error message — it usually tells you what's wrong. You can also paste the error to Claude and ask what it means.
